Keygrain

Deterministic password derivation. One secret, all your passwords.

Enter your master secret + email → get a unique, strong password for any site. Same inputs always produce the same output.

How it works

  • No password storage — passwords are recomputed on demand
  • Change length, symbols, or salt → entirely different password
  • Encrypted config backup (optional)
